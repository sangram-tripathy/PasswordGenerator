# PasswordGenerator

Developed a responsive password generator web application using React functional components and Hooks (useState, useEffect, useCallback, useRef).

Implemented dynamic password generation with configurable length, numbers, and special characters.

Used Clipboard API to enable one-click password copy functionality.

Ensured real-time password updates based on user input using React state management.

Designed a clean and user-friendly UI with Tailwind CSS.

Improved performance by memoizing functions with useCallback.
